# Pandamonium Gaming Community

A member-driven gaming community guided by core values of **inclusivity, equity, and accountability**.

Founded in 2004, Pandamonium fosters a safe and drama-free environment where gamers can forge lasting connections while treating each other with respect.

## 🎮 About

Pandamonium is a gaming community that brings together players across multiple games including:
- Dune: Awakening
- Palia
- Final Fantasy XIV Online
- And more!

## 🛠️ Technology

This website is built with modern web technologies:

- **React 19** — Modern UI framework
- **Tailwind CSS 4** — Utility-first styling
- **Wouter** — Lightweight client-side routing with hash-based navigation
- **Framer Motion** — Smooth animations and transitions
- **Vite** — Lightning-fast build tool
- **GitHub Pages** — Reliable hosting
- **Cloudflare** — Security, performance, and custom headers

### Key Features

- ✅ Hash-based routing for GitHub Pages compatibility
- ✅ Comprehensive Content Security Policy (CSP)
- ✅ Security headers (X-Frame-Options, X-Content-Type-Options, Referrer-Policy)
- ✅ Responsive design for all devices
- ✅ Accessibility-first approach (ARIA labels, keyboard navigation)
- ✅ Performance optimized (lazy loading, async image decoding)
